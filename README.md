# Vive Servicios

Vive Servicios es un aplicativo web desarrollado con HTML, CSS y JavaScript, que permite explorar servicios digitales en formato de catálogo.

La aplicación ofrece una experiencia de usuario sencilla e intuitiva, permitiendo visualizar servicios, ver detalles, gestionar favoritos y utilizar formularios de contacto y registro.

---

## Funcionalidades

- Visualización de servicios en catálogo
- Vista detallada de cada servicio
- Sistema de favoritos
- Formulario de registro de usuarios
- Formulario de contacto
- Navegación sencilla e intuitiva

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
